# Style Transfer Web

AI Image Style Transfer Tool - MVP Version

## Features
- Upload image (drag & drop)
- Select art style (5 presets)
- AI-powered style transfer
- Download result

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

## API Configuration
Create `.env.local`:
```
NEXT_PUBLIC_API_URL=your_api_url
NEXT_PUBLIC_API_KEY=your_api_key
```
